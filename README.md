# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes telecom customer behavior to understand customer churn patterns and identify factors influencing retention.

The objective is to transform raw customer data into business insights using Python and Power BI.


## Business Problem

Customer churn affects business growth and revenue.

This project answers:

* Which customers are leaving?
* Which contract type has highest churn?
* How does revenue relate to churn?
* What factors improve retention?


## Dataset

IBM Telco Customer Churn Dataset

Rows: ~7000 customers
Columns: 21

Target Variable:

* Churn

## Key Insights

* Month-to-month contracts showed higher churn.
* Customers with lower tenure were more likely to leave.
* Higher monthly charges were associated with churn.
* Long-term contracts generated greater revenue.
